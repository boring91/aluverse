# AluVerse

TanStack Start application for AluVerse. The repo uses Bun for package management and scripts.

## Development

```bash
bun install
bun run dev
```

`bun run dev` starts the app and local infrastructure through `mprocs`.

## Quality Checks

```bash
bun run lint
bun run check
bun run build
```

## Database

```bash
bun run db:generate
bun run db:migrate
bun run db:seed
bun run db:studio
```

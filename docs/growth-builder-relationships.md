## Ideal Builder Profile

### Primary Targets

**Tier 1: Volume Home Builders**

- Building 10-50 homes annually
- Focus on townhouses, duplexes, multi-unit
- Project size: $3,000 - $15,000 per unit
- Examples: AAA Bayside Homes, Crownview Projects

**Tier 2: Boutique Custom Home Builders**

- Building 3-10 custom homes annually
- High-end specifications
- Project size: $10,000 - $50,000 per home
- Higher margins, longer relationships

**Tier 3: Renovation Specialists**

- Focus on knockdown-rebuilds
- Heritage conversions
- Project size: $5,000 - $20,000

### Geographic Focus

Based on current project distribution, prioritize builders in:

1. **Sutherland Shire** - emerging growth area (P0044, P0047, P0048)
2. **Eastern Suburbs** - strong existing presence
3. **Inner West** - redevelopment corridor
4. **Hills District** - new development area

### Red Flags to Avoid

- Payment terms beyond 45 days
- No established track record
- Price-shopping every project
- Poor communication/unrealistic timelines

## Relationship Management

### Onboarding New Builders

**Week 1:**

- Site visit to assess first project
- Detailed quote with specifications
- Introduction to key team members

**Week 2-4:**

- Complete first project
- Follow-up call within 48 hours of completion
- Request feedback

**Month 2:**

- Check in on upcoming projects
- Ask for referrals
- Review pricing structure

### Ongoing Management

**Monthly:**

- Check-in call/email
- Share any new product offerings
- Confirm upcoming project pipeline

**Quarterly:**

- In-person meeting if possible
- Review relationship health
- Discuss any issues or improvements
